前言

本文将学习创建一个应用来访问RESTFul Web Service。

你将会创建

你将会创建一个使用Spring的RestTemplate来获取Spring Boot的RESTFul服务 http://gturnquist-quoters.cfapps.io/api/random中的随机返回信息的应用。

你将需要

  • 大约15分钟
  • 一个熟悉的IDE
  • JDK 1.8以上
  • MAVEN 3.0以上
